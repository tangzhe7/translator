# 贡献须知

### 领取原则
```
1.采用领取的原则，领取前先提ISSUE，说明准备翻译的文档和范围，避免出现重复翻译。
  如果已经有人翻译了，你也可以重复翻译，此时不用提ISSUE，当然，你翻译完后可以提PR，
  我们会校验，如果更好，会通过。
```
### 文件格式要求
```
2.接受 markdown, word, pdf格式（如果是连续章节的，建议使用markdown）
```
### 译稿格式
```
1.译稿内头俩行使用如下格式
  原文:[这是一个超链接-原文链接]
  译者:名字（你的邮箱）
```
```
2.翻译的原始文档，如果是PDF，请把原始PDF一起上传。
```
